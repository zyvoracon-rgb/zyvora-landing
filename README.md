# Zyvora Landing

Production-ready static landing page for the Zyvora digital product.

## Files

- `index.html` - Semantic HTML5 landing page with SEO and Open Graph metadata.
- `style.css` - Mobile-first CSS3 with variables, responsive layout, hover states, and animations.
- `script.js` - Vanilla JavaScript for sticky navigation state, mobile menu, reveal animations, counters, and FAQ accordion.
- `assets/` - Local SVG icons, favicon, and social preview artwork.

## Structure

```text
zyvora-landing/
├── assets/
│   ├── icons/
│   └── images/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Performance Notes

- Vanilla HTML, CSS, and JavaScript only.
- No Bootstrap, Tailwind, React, or external UI libraries.
- SVG assets are local and lightweight.
- CSS and JavaScript are small, dependency-free, and optimized for fast loading.
- Animations respect `prefers-reduced-motion`.

## Customization

Brand values are centralized in `style.css`.
